# 第六类：ECS System测试

Bevy特色。

---

例如：

```rust

```

---

直接建一个 App。

```rust

```

---

加入：

```rust

    Update,
    apply_damage_system
);
```

---

创建实体：

```rust
app.world_mut().spawn((
    Health(100),
));
```

---

运行：

```rust
app.update();
```

---

检查：

```rust
assert_eq!(hp, 80);
```

---

这是Bevy官方推荐思路。

---

# 示例

```rust
#[test]
fn damage_system_works() {

    let mut app = App::new();

    app.add_systems(
        Update,
        apply_damage_system
    );

    let entity =
        app.world_mut()
            .spawn((
                Health(100),
                PendingDamage(20),
            ))
            .id();

    app.update();

    let hp =
        app.world()
            .get::<Health>(entity)
            .unwrap();

    assert_eq!(hp.0, 80);
}
```
