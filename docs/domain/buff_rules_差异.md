### 2. 术语定义新增 ModifierSource（新增）

**位置**：术语定义

**变更类型**：新增

**新内容**：
```
## ModifierSource

修饰符来源标识，用于追踪修饰符归属。

不是修饰符本身。ModifierSource 是来源标识，Modifier 是属性修饰。

关键属性：
- buff_source(instance_id)：Buff 来源
- trait_source(trait_id)：Trait 来源
- equipment_source(slot)：装备来源
```

**原因**：ModifierSource 是修饰符追踪的核心概念，原文档在规则和管线中提及但未在术语中定义。
