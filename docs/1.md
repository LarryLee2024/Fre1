## 使用说明
1. 将上述 JSON 保存为 `test_cases/combat_basic.json`
2. 配合 `test_spec_v2.0.schema.json` 使用，VS Code 等编辑器会自动提供语法验证和自动补全
3. 可以通过简单的脚本将此 JSON 直接编译为 Rust 测试代码：
    ```rust
    // 示例生成逻辑
    fn generate_test_case(test_case: &TestCase) -> String {
        format!(
            r#"
            #[test]
            fn test_{}() {{
                // {}
                {}
                {}
                {}
            }}
            "#,
            test_case.id.to_lowercase().replace("-", "_"),
            test_case.title,
            generate_preconditions(test_case.preconditions),
            generate_steps(test_case.steps),
            generate_assertions(test_case.assertions)
        )
    }
    ```
4. 可以集成到 CI 流程中，自动验证所有测试用例是否符合规范
