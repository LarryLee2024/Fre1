## Feature Flags

以下是运行示例所需启用的 Cargo feature flag 说明：

| Feature | 用途 |
|---------|------|
| `debug` | 调试可视化(网格线/坐标轴/碰撞体) |
| `text` | Tiled 文本对象支持 |
| `avian` / `avian_debug` | Avian2D 物理后端 |
| `rapier` / `rapier_debug` | Rapier2D 物理后端 |
| `physics` | 自定义物理后端 |
| `user_properties` | Tiled 自定义属性→Bevy 组件映射 |
