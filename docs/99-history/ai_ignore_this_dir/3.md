# 最大缺陷1

缺少：

```text
架构漂移（Architecture Drift）
```

---

例如：

ADR规定：

```text
Character
↓
Ability
↓
Effect
```

依赖方向。

---

半年后：

```rust
effect
↓
use character::xxx
```

出现。

---

编译没问题。

测试没问题。

Dead Code也没问题。

---

但：

```text
架构已经腐化
```

---

应该增加：

## 5. Architecture Drift

扫描：

```text
ADR依赖方向
实际依赖方向
```

差异。

例如：

```text
Debt-ADR-001
Ability 不应依赖 Tactical

Debt-ADR-007
Effect 不应依赖 Character
```

---

这个非常重要。

大型项目后期最常见。

---

# 最大缺陷2

缺少：

```text
抽象泄漏（Abstraction Leakage）
```

例如：

你最近就在处理：

```rust
TagSet
AttributeContainer
ModifierContainer
```

泄漏。

---

实际上：

```text
Capabilities
Character
Ability
Effect
```

都会发生。

---

应该单独一类：

## Abstraction Leakage

扫描：

```rust
use xxx::internal
```

```rust
use xxx::mechanism
```

```rust
use xxx::model
```

跨领域出现。

---

而不是仅限：

```text
Capabilities类型泄漏
```

---
