## Naming

Type      -> PascalCase
Trait     -> Verb/Capability
Function  -> snake_case
Constant  -> SCREAMING_SNAKE_CASE

### 测试函数命名
测试函数名用**中文**，技术术语如 UI/AI/HP/MP/Buff 等保留英文：
```rust
#[test]
fn 物理伤害正确应用护甲减免() { ... }

#[test]
fn Buff到期自动移除() { ... }
```

## File

One file = one topic

Avoid:
- systems.rs
- components.rs
- utils.rs

## Function

- Single responsibility
- Early return
- Max nesting <= 3

## Module

- private by default
- expose minimal API

## Rust

- avoid clone()
- avoid unwrap()
- prefer iterator
- prefer Result
