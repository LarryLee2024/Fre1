## Naming

Type      -> PascalCase
Trait     -> Verb/Capability
Function  -> snake_case
Constant  -> SCREAMING_SNAKE_CASE

### 测试函数命名
测试函数名用英文 snake_case，使用业务术语如 `damage_applies_armor_reduction`、`buff_removed_on_expiry`：
```rust
#[test]
fn damage_applies_armor_reduction_correctly() { ... }

#[test]
fn buff_removed_on_expiry() { ... }
```

## File

One file = one topic

Avoid:
- systems.rs
- components.rs
- utils.rs

Note: Domain 内部按需拆分，详见 `ai-constitution-complete.md` §3.4

## Function

- Single responsibility
- Early return
- Max nesting <= 3

## Module

- private by default
- expose minimal API

## Rust

- avoid clone()
- avoid unwrap()
- prefer iterator
- prefer Result
