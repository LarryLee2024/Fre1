## Naming

Type      -> PascalCase
Trait     -> Verb/Capability
Function  -> snake_case
Constant  -> SCREAMING_SNAKE_CASE

## File

One file = one topic

Avoid:
- systems.rs
- components.rs
- utils.rs

## Function

- Single responsibility
- Early return
- Max nesting <= 3

## Module

- private by default
- expose minimal API

## Rust

- avoid clone()
- avoid unwrap()
- prefer iterator
- prefer Result
